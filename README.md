# Traffic Light Controller (FSM) - Verilog

This project implements a Traffic Light Controller using a Finite State Machine (FSM) in Verilog HDL.

States:
GREEN
YELLOW
RED

State transitions:
GREEN → YELLOW → RED → GREEN

Simulation Environment:
EDA Playground / Icarus Verilog

Files:
traffic_light.v - FSM design
tb_traffic_light.v - Testbench
waveform.png - Simulation waveform result
